HTML Semantic Tags Reference

A comprehensive guide to all semantic HTML elements

Document Structure

<header>

Introductory content or navigation for a section/page

<nav>

Navigation links section

<main>

Main content of the document (only one per page)

<section>

Thematic grouping of content, typically with a heading

<article>

Self-contained, independently distributable content

<aside>

Tangentially related content (sidebars, callouts)

<footer>

Footer information for a section/page

Text Content

<h1> to <h6>

Headings in hierarchical order

<p>

Paragraph

<blockquote>

Extended quotation from another source

<cite>

Reference to a creative work

<q>

Short inline quotation

<code>

Code snippet

<pre>

Preformatted text

<mark>

Highlighted or marked text

<time>

Specific time or date

<address>

Contact information

Lists

<ul>

Unordered list

<ol>

Ordered list

<li>

List item

<dl>

Description list

<dt>

Term in a description list

<dd>

Description/definition in a description list

Embedded Content

<figure>

Self-contained content like images, diagrams, code

<figcaption>

Caption for a figure

<audio>

Sound content

<video>

Video content

<picture>

Container for multiple image sources

Forms & Interactive

<form>

Interactive form

<fieldset>

Group of related form controls

<legend>

Caption for a fieldset

<label>

Label for a form control

<output>

Result of a calculation

Tables

<table>

Tabular data

<thead>

Header rows group

<tbody>

Body rows group

<tfoot>

Footer rows group

<caption>

Table caption

Other Semantic Tags

<details>

Disclosure widget that can be toggled open/closed

<summary>

Summary/caption for details element

<dialog>

Dialog box or modal

<data>

Links content with a machine-readable value

<abbr>

Abbreviation or acronym

<strong>

Strong importance (typically bold)

<em>

Emphasized text (typically italic)

<small>

Fine print or side comments

<del>

Deleted/removed text

<ins>

Inserted/added text

<s>

Strikethrough (no longer accurate/relevant)

<sub>

Subscript

<sup>

Superscript

<kbd>

Keyboard input

<samp>

Sample output from a program

<var>

Variable in programming/math context
